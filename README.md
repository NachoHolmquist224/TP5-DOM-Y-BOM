# 🌐 Trabajo Práctico N°5: DOM & BOM

Este repositorio está dedicado al estudio y manipulación de la estructura de documentos web (**DOM**) y la interacción con las facultades del navegador (**BOM**). El objetivo es crear experiencias dinámicas que reaccionen a las acciones del usuario.


### 🧪 Conceptos Centrales

#### 🔹 DOM (Document Object Model)
Representación del documento HTML como un árbol de nodos, permitiendo:
* **Selección de elementos:** `querySelector`, `getElementById`, `getElementsByClassName`, etc...
* **Manipulación:** Creación y eliminación de nodos, cambio de estilos y clases.
* **Eventos:** Escucha y manejo de interacciones (clicks, submit, etc...).

#### 🔹 BOM (Browser Object Model)
Interacción con el navegador fuera del documento:
* **Window:** Control de dimensiones, alertas y timers (`setTimeout`, `setInterval`).
* **Location & History:** Gestión de URLs y navegación.
* **Navigator:** Datos del sistema y permisos del usuario.
